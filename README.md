机械臂夹爪控制器  
-----------------------------

1. 初始化:  
   - 创建与机械臂的连接，IP地址"192.168.1.18"，端口8080  
   - 设置通信参数（Modbus模式，波特率115200）  

2. 主要功能:  
   - activate(): 唤醒夹爪（发送初始化指令）  
   - goToPosition(): 移动夹爪至目标位置并等待到达  
   - getPosition(): 读取当前夹爪位置（0-255范围）  
   - stop(): 立即停止夹爪运动  

核心特性:  
- 阻塞式运动（等待直至到达目标位置）  
- 实时位置监控  
- 安全停止功能  
- 通过Modbus直接硬件通信
